CliColor — ANSI 256-color reference

This file is generated from the same color table that defines the CliColor enum (ANSI index + name) and CliColorPalette (RGB). Values 0–15 preserve the standard ConsoleColor-compatible names and ordering. Values 16–255 map directly to ANSI 256-color palette indexes: 16–231 are the 6×6×6 color cube and 232–255 are the grayscale ramp. The enum numeric value always equals the ANSI index. Markup can use any of these names directly, e.g. [OceanBlue], [OrangeRed], [Gray85].
