Select prompt — initial state

The dialog frames the list, the first row is active, and the hint row documents the keys. Escape cancels and returns null.

string? environment = await TigerTui.SelectAsync(
    "Choose environment",
    ["Local", "Demo", "Staging", "Production"]);

Multi-select prompt — initial state

Each row carries a checkbox; "Logging" was preselected by the caller. The hint row documents toggling and the bulk keys.

int[]? indexes = await TigerTui.MultiSelectIndexesAsync(
    "Choose features",
    ["Logging", "Metrics", "Trace", "Audit"],
    preselectedIndexes: [0]);
