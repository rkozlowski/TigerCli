Getting started — ROI Cities

The two example apps from docs/getting-started.md, captured from real runs and real rendering with the TigerBlue theme: the basic app's script-safe shape, and the extended app's provider-backed prompting, command menu, and typed exit codes.

Basic — generated help (--help)

Metadata-driven help for the basic app: description, the two commands, and the built-in framework options.

roi-cities --help
roi-cities
  Cities of the Republic of Ireland.

Usage:
  roi-cities <command> [options]

Commands:
  list  Lists the cities.
  show  Shows details for one city.

Options:
  -h, --help
      Show help
  --help-errors
      Show exit-code help
  --non-interactive
      Disable interactive prompts
  --theme <theme>
      Select the UI theme by name
      dark | light | tiger-blue
  --culture <culture>
      Select UI culture for framework strings

Basic — a missing city is prompted

Run semi-interactively with the city argument missing, show asks for it with a text input instead of failing.

roi-cities show
╔══════════════════════════════════════════════════════════════╗                                                        
║City name, e.g. Galway.                                       ║                                                        
║                                                              ║                                                        
╚══════════════════════════════════════════════════════════════╝                                                        
A value is required.

Basic — --non-interactive fails cleanly

The same missing argument under --non-interactive: no prompt, an error on stderr, and a non-zero exit code — automation never hangs on a question.

roi-cities show --non-interactive
Error: Missing required argument: <city>

Extended — generated help (--help)

The extended app's help adds the display name, version metadata, and the exit-code hint from the typed exit-code registration.

roi-cities --help
roi-cities
  Cities of the Republic of Ireland.

Usage:
  roi-cities [options]
  roi-cities <command> [options]

Commands:
  list  Lists the cities.
  show  Shows details for one city.

Options:
  -h, --help
      Show help
  --version
      Show version
  --version-full
      Show product version
  --help-errors
      Show exit-code help
  --non-interactive
      Disable interactive prompts
  --theme <theme>
      Select the UI theme by name
      dark | light | tiger-blue
  --culture <culture>
      Select UI culture for framework strings

For a list of exit codes, use --help-errors.
