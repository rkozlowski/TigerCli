folder-copy
  Recursively copies the contents of one folder into another.

Usage:
  folder-copy [options]

Options:
  -s, --source <value>
      Source folder to copy from.
      Required.
  -d, --destination <value>
      Destination folder to copy into.
      Required.
  -h, --help
      Show help
  --help-errors
      Show exit-code help
  --non-interactive
      Disable interactive prompts
  --theme <theme>
      Select the UI theme by name
      dark | light | tiger-blue
  --culture <culture>
      Select UI culture for framework strings
