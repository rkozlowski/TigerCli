parser-test
  TigerCli command parser manual test app.

Usage:
  parser-test [options]
  parser-test <command> [options]

Commands:
  echo     Echoes a message.
  mode     Prints the effective interaction mode.
  features Selects feature flags with a dynamic multi-select.
  fail     Returns a typed failure code.
  throw    Throws an exception to test UnhandledException mapping.
  raw      Returns a raw integer exit code.
  projects
  prompt
  provider

Options:
  -n, --name <value>
      Name to greet.
  -h, --help
      Show help
  --help-errors
      Show exit-code help
  --non-interactive
      Disable interactive prompts
  --theme <theme>
      Select the UI theme by name
      dark | light | tiger-blue
  --culture <culture>
      Select UI culture for framework strings

For a list of exit codes, use --help-errors.
