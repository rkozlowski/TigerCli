# ItTiger.TigerCli

An opinionated .NET CLI/TUI framework for command-driven applications, with commands, prompts, providers, semantic rendering, structured output, testing support, and script-safe interaction modes.

```bash
dotnet add package ItTiger.TigerCli --version 0.8.1
```
