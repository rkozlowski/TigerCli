# libwebp tools (local, not committed)

`img2webp.exe` assembles the animated spinner previews under `docs/examples/spinners/`
(see `internal/DocSamples`, `spinners` mode). The binary is not committed; fetch it locally:

1. Download the precompiled Windows x64 package from
   <https://developers.google.com/speed/webp/docs/precompiled>
   (direct: <https://storage.googleapis.com/downloads.webmproject.org/releases/webp/libwebp-1.6.0-windows-x64.zip>).
2. Extract `bin/img2webp.exe` into this directory: `internal/tools/libwebp/img2webp.exe`.

Version used for the committed artifacts: **libwebp 1.6.0** (WebP Encoder 1.6.0).
`COPYING` from the same package carries the libwebp license.
